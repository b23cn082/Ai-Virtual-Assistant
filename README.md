# AI Virtual Assistant

## Project Overview

AI Virtual Assistant is a Python-based desktop assistant that recognizes voice commands and performs tasks such as opening websites, searching Wikipedia, telling jokes, and providing the current date and time.

---

## Features

- Voice Recognition
- Text-to-Speech
- Open Google
- Open YouTube
- Open Gmail
- Open GitHub
- Open ChatGPT
- Wikipedia Search
- Google Search
- Tell Time
- Tell Date
- Tell Jokes
- Exit Command

---

## Technologies Used

- Python
- SpeechRecognition
- PyAudio
- pyttsx3
- Wikipedia API
- Webbrowser Module

---

## Installation

```bash
pip install -r requirements.txt
```

Run:

```bash
py main.py
```
